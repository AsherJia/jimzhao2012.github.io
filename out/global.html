tagname="init_member_H5_info"()

进入H5模块，初始化数据 H5接收到web_view_did_finished_load的回调之后，调用该函数，初始化数据会通过callback传递给H5
Since:
  • version 5.2
Author:
  • jimzhao
Source:
  • /Users/jimzhao/Desktop/md/Hello/bridge.js, line 260
Example
CtripUtil.app_init_member_H5_info();
//调用完成，H5页面会收到如下返回数据
var json_obj =
{
   tagname:"init_member_H5_info",
   timestamp:135333222,
   version:"5.2",
   device:"iPhone4S",
   appId:"com.ctrip.wrieless",
   serverVersion:"5.3",
   platform:1, //区分平台，iPhone为1, Android为2
   userInfo={USERINFO},//USERINFO内部结构参考CtripUser.app_member_login();    
}
app.callback(json_obj);
