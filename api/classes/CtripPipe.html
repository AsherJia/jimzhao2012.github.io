app_send_H5_pipe_request

(
  • serviceCode
  • header
  • data
  • sequenceId
)

定义在 bridge.js:1637

可用版本 v5.4

H5通过App发送服务

参数:

  • serviceCode String

    需要发送服务的服务号

  • header String

    服务的header

  • data String

    服务所需要的数据部分，各个服务都不同

  • sequenceId String

    发送服务的序列号，随机生存即可

举例:

 CtripUtil.app_send_H5_pipe_request("9500001", "H5Agent","{}");
//调用后，H5会收到native回调的数据
   var json_obj =
   {
       tagname:"send_h5_pipe_request",
       param:
       {
           sequenceId:"13523333333",
           pipeResponse:"eHh4eHh4",
       },
   }
   app.callback(json_obj);

app_send_HTTP_pipe_request

(
  • target
  • method
  • header
  • queryData
  • retryInfo
)

定义在 bridge.js:1606

可用版本 v5.4

H5通过App发送服务

参数:

  • target String

    HTTP请求发送的URL地址

  • method String

    HTTP请求方式GET/POST

  • header String

    HTTP头，JSON格式，cookie作为一个key存储再HEADER内部

  • queryData String

    GET请求时候，会将该字符串append到URL之后，POST请求，会将queryData UTF8编码后，作为HTTPBODY post

  • retryInfo String

    重试相关信息，JSON格式，以下3个key：timeout, retry, retryInterval

举例:

 CtripUtil.app_send_H5_pipe_request("9500001", "H5Agent","{}");
//调用后，H5会收到native回调的数据
   var json_obj =
   {
       tagname:"send_h5_pipe_request",
       param:
       {
           pipeResponse:"eHh4eHh4",
       },
   }
   app.callback(json_obj);
